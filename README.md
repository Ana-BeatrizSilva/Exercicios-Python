# 🐍 Exercícios de Python (Iniciante)

Este repositório contém 16 exercícios desenvolvidos em Python com foco na prática da lógica de programação.

## 📚 Conteúdo abordado:

Os exercícios incluem:

* Entrada e saída de dados (`input` e `print`)
* Operações matemáticas básicas
* Cálculo de média
* Conversões de unidades (temperatura, medidas e armazenamento)
* Cálculo de áreas geométricas
* Estruturas condicionais simples
* Cálculos do dia a dia (salário, multas, consumo de tinta)

## 📝 Lista de exercícios:

1. Exibir mensagem "Olá, Mundo!"
2. Exibir um número informado
3. Soma de dois números
4. Média de quatro notas
5. Conversão de metros para centímetros
6. Cálculo da área de um círculo
7. Área de um retângulo e seu dobro
8. Cálculo de salário mensal
9. Conversão de Fahrenheit para Celsius
10. Conversão de Celsius para Fahrenheit
11. Operações com números inteiros e reais
12. Conversão de Gigabytes para Megabytes
13. Conversão de Gigabytes para Kilobytes
14. Cálculo de multa por excesso de peso
15. Cálculo de salário com descontos
16. Cálculo de tinta necessária para pintura de uma área

## 🎯 Objetivo:

Este projeto foi desenvolvido para fins de estudo, com intuito de praticar conceitos fundamentais da linguagem Python e desenvolver o raciocínio lógico através de exercícios simples e progressivos.
